# Python Assignment — Part 1: Python Basics & Control Flow

## Overview
A command-line **Student Grade Tracker** demonstrating core Python concepts.

## Tasks
| Task | Description | Marks |
|------|-------------|-------|
| 1 | Data Parsing & Profile Cleaning | 5 |
| 2 | Marks Analysis using Loops & Conditionals | 8 |
| 3 | Class Performance Summary | 7 |
| 4 | String Manipulation Utility | 5 |

## How to Run
```bash
python part1_grade_tracker.py
```
> Task 2 includes an interactive while-loop prompt. Type a subject name and marks, then type `done` to finish.

## Requirements
- Python 3.8+
- No external libraries needed
